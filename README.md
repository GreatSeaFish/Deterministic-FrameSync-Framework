# 🚀 Deterministic FrameSync Framework for SLG

一个专为大地图 SLG / RTS 游戏设计的**纯 C# 确定性定点数帧同步联机框架**。本项目以 Godot 4.x (.NET 8) 作为渲染前端进行演示，其核心状态机与网络层完全解耦，可独立抽取用于任何 C# 游戏引擎中。

---

## 🚀 快速上手 (Quickstart)
想要在本地一键跑通联机演示，只需以下三步：

1. 克隆项目与依赖检查
确保你本地已安装 .NET 8 SDK 以及支持 .NET 的 Godot 4.x (Mono/C# 版本)。

2. 启动帧同步独立服务端
框架的服务端是一个纯粹的 .NET 控制台程序，无需运行 Godot 即可独立启动：

Bash
cd Server
dotnet run
服务端启动成功后将提示：[Server] 服务端已启动，最大支持 8 名玩家... 并在 5721 端口监听
3. 运行 Godot 客户端（多端测试）
用 Godot 打开项目根目录的 project.godot。

运行项目（默认切入 page_login.tscn 登录）。

本地多端联机测试方法：

开启第一个客户端，输入昵称 玩家A，IP 保持默认的 127.0.0.1:5721，点击加入游戏进入大厅。

选择你的房间位置（位置1-8）和代表颜色，勾选准备。

保持第一个客户端运行，开启第二个客户端（或通过 Godot 运行第二个实例），输入昵称 玩家B 进入大厅。

当至少 2 名玩家加入且全员点击准备后，服务端自动开启逻辑帧心跳，两端同时切入主舞台 WorldRender.tscn。

操作：使用 WASD 键移动，你将在两个客户端屏幕上看到基于定点数严格同步的方块位移。

## 🛠️ 核心技术亮点

* **🛡️ 纯主线程驱动 · 告别多线程噩梦**
  彻底重构传统的独立网络线程架构，将 LiteNetLib 的物理事件轮询 (`PollEvents`) 与发包时钟完全挂载在 Godot 的主线程生命周期中，天然杜绝死锁与多线程竞态条件（Race Conditions）。
* **📐 强确定性（Deterministic）状态机**
  集成 `FixedMath.NET` 定点数库，所有逻辑坐标位移完全脱离渲染层 Node 节点，纯内存定点数数学运算，从根本上消除了不同 CPU 架构与编译环境下的浮点数精度漂移。
* **🧠 智能抖动缓冲区 (Jitter Buffer) 与弹性追帧算法**
  日常状态下进行首次攒帧平滑驱动；轻微抖动时触发**温和追帧**（单周期双步步进）；切后台或遭遇严重断连恢复后，自动触发**激进全速追帧**，瞬时消化积压帧，保证多端强同步。
* **📡 “静止沉默机制”压低带宽**
  客户端发包器内置静止防御，当玩家处于松开键盘、无指令输入的沉默状态时，框架自动停止物理发包，大幅压低高频帧同步框架在 SLG 大地图上的日常网络带宽开销。
* **📅 内置 SLG 级联历法结算引擎**
  状态机时钟深度整合了确定性时钟 `GameTime`。逻辑帧推进可直接级联驱动 `SettlementManager` 的日结算与跨月业务结算（处理如每日资源产出、每月军队维护费扣除等深层 SLG 业务）。
* **🗂️ 健壮的自定义二级制协议路由**
  基于纯字节包头（PacketType 1-byte Header）对齐机制，全自动实现大厅状态机（房间插槽抢占、颜色互斥、全员准备开局）到游戏内 FrameData 逻辑帧的无缝切换。

## 🗂️ 目录结构树

```text
ParadoxSimulator/
├── Core/               # ⚡ 客户端逻辑驱动中心（核心解耦资产）
│   ├── ClientCommandSender.cs   # 输入采集与静止沉默发包器
│   ├── ServerCommandHandler.cs  # 抖动缓冲区与弹性追帧控制器
│   └── GameTime.cs              # 确定性时钟步进管理器
├── Shared/             # 🌐 动静分离的跨端共享库（服务端/客户端共用）
│   ├── Math/                    # FixVector2 确定性定点数向量
│   └── Protocol/                # 基于 INetSerializable 的二进制协议包
├── Server/             # 🖥️ .NET 8 控制台独立服务端（纯C#驱动）
├── Render/             # 🎨 Godot 渲染/UI前端视角（大厅页面、像素映射）
└── CoreHost.cs         # 以 godot 为渲染器示例的程序入口，挂载AutoLoad
```
